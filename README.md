# Nate B. Jones - AI Career Strategist Landing Page

A conversion-focused landing page built for [Nate B. Jones](https://www.instagram.com/nate.b.jones/) (@nate.b.jones), AI career strategist and former Amazon Prime Video Head of Product.

**Context:** This was a real project for a real client. I built this as part of a technical interview process and qualified for the final rounds as a vibecoder with no formal programming background.

## What it does

Single-page scroll design with progressive disclosure pattern guiding visitors from free resources to premium enterprise services:

- Hero section with value proposition
- Substack newsletter integration (live API)
- Tiered service offerings (Explore, Learn, Accelerate, Enterprise)
- Testimonials and social proof
- Newsletter signup and lead capture

## Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript (strict)
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React
- **Deployment:** Vercel

## Built with

Built entirely with AI-assisted development (Windsurf IDE at the time, before I moved to Claude Code for all development). This project represents an early stage of my vibecoding journey.
